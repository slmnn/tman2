Tällä sivulla voit muuttaa käyttäjän tietoja. Pääset muokkaustilaan painamalla työkalu-kuvaketta.

Käyttäjän sähköpostiosoite tulee olla yksilöllinen.

Jos tunnus on liitetty alueomistajaan, käyttäjä voi muuttaa ainoastaan oletusomistajalla (Aluepöytä) ja itsellään olevia alueita. Alueomistajaan liittyvä tunnus ei voi olla pääkäyttäjä. Voit poistaa linkityksen alueomistajaan valitsemalla linkitetyksi omistajaksi oletusomistajan (Aluepöytä).

Voit antaa käyttäjälle lisäoikeuksia rastittamalla "Voi ..." -alkuiset oikeudet. Rastien poistaminen poistaa oikeudet.

Väärin luotua tunnusta ei voi poistaa, se voidaan vain arkistoida. Arkistoitu käyttäjä ei voi kirjautua sovellukseen.
