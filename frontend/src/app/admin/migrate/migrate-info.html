Tällä sivulla voit tehdä migraation vanhasta aluesovelluksesta uuteen.

  1. Tee ensin sovelluksen oletusomistaja (Aluepöytä) ja aseta se sovelluksen asetuksissa oletusomistajaksi.
  2. Kopioi varmuuskopiosta alueomistajadata ylempään tekstiruutuun. Paina ko. ruudun alta "Tallenna".
  3. Kopioi varmuuskopiosta aluedata alempaan tekstiruutuun. Paina alemman ruudun alta "Tallenna"
  4. HUOM! Tarkista tietokannassa, että luontitieto on oikein tehdyillä alueilla ja alueomistajilla. Katso alta hyödyllisiä komentoja.
  • Kantaan: 
    use dbname
  • Näytä taulut: 
    show collections
  • Näytä käyttäjät: 
    show users
  • Haku: 
    db.collectionname.find({kenttä:"arvo"})
  • Hae alueet joilla luontitieto pielessä: 
    db.territory.find({createdId:null})
  • Omistajan päivittäminen: 
    db.territory.update({holder: null}, { $set: { holder: ObjectId("56667e44c75ab9f1eed261a5")}}, false, true);
